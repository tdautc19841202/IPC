ipcam 产测网络通信协议

udp 端口 8313 纯文本通信协议

发现设备：
广播       uid?
设备端应答 uid:xxxxx

设置 uid:
发送       uid=xxxxx
设备端应答 uid.

查询 sid：
广播       sid?
设备端应答 sid:xxxxx

设置 sid:
发送       sid=xxxxx
设备端应答 sid.

获取版本号：
发送       ver?
设备端应答 ver:xxxxx

喇叭咪头自动测试：
发送       micspktest!
设备端应答 micspktest.

查询测试结果：
发送       result?
设备端应答 result:xxxxx
 - 第一个 x 表示 micspk 测试结果
 - 第二个 x 表示按键测试结果
 - 第三个 x 表示光感测试结果
 - 第四个 x 表示 sd 测试结果
 - 第五个 x 表示灯板串口硬件测试结果
 x 的值可以为：- 表示测试结果未知，y 表示测试通过，n 表示测试失败

查询 wifi 信号强度：
发送       wifi_signal?
设备端应答 wifi_signal:xxx

开启 iperf 网络吞吐量测试：
发送       iperf3=x （x 为 0 表示关闭，1 表示打开）
设备端应答 iperf3.

查询 ft_uid:
发送       ft_uid?
设备端应答 ft_uid:xxxxx
